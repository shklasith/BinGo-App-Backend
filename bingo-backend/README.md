# 🎰 Bingo App - Backend

Welcome to the backend service for the **Bingo Application**, developed as part of the **MAD Group Project**.  
This backend handles the core logic, API endpoints, authentication, and database operations.

---

## 📌 Overview

The backend is responsible for managing:

- 🎮 Bingo game sessions  
- 👤 User authentication  
- 📊 Game data and results  
- 💾 MongoDB database operations  
- 🔐 Secure API requests  
- 🌐 Frontend communication via REST APIs  

---

## 🛠️ Tech Stack

| Technology | Description |
|------------|-------------|
| Runtime | Node.js / Bun |
| Language | TypeScript |
| Database | MongoDB |
| Package Manager | Bun *(Recommended)* |
| API | REST API |
